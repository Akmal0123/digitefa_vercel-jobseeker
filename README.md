
# DIGITEFA

Run Localy on Frontend side.

## Langkah-langkah untuk Menjalankan Aplikasi

### 1. Clone Proyek

Clone repositori DIGITEFA dari GitHub:

```bash
git clone https://gitlab.com/job-portal9594324/job-portal-clientside-cms.git
```

### 2. Masuk ke Direktori Proyek

Pindah ke direktori proyek DIGITEFA yang telah di-clone:

```bash
cd cms-clientside
```

### 3. Instalasi Dependensi

Install semua paket dan dependensi yang diperlukan menggunakan npm:

```bash
npm install
```

### 4. Menjalankan Server

Jalankan server pengembangan lokal untuk proyek DIGITEFA:

```bash
npm run dev
```
